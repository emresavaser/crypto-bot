# Claude Code Proje Ayarları

## Proje: CryptoBot - Kripto Al-Sat Botu

### Proje Tipi
- Full-stack kripto trading bot
- Frontend: Next.js 15 + TypeScript + Tailwind CSS
- Backend: Python FastAPI

### Önemli Dosyalar
- `claude.md` - Detaylı dokümantasyon
- `PROJE_DURUMU.md` - Mevcut durum ve yapılacaklar
- `crypto-bot-ui/` - Next.js frontend
- `bot-backend/` - Python backend

### Çalıştırma
```bash
# Frontend
cd crypto-bot-ui && npm run dev

# Backend
cd bot-backend && py main.py
```

### Bağımlılıklar
- Frontend: Yüklü (node_modules mevcut)
- Backend: Yüklü (pip install yapıldı)

### Son Durum
- Frontend çalışıyor (localhost:3000)
- Backend yapısı hazır, başlatılmadı
